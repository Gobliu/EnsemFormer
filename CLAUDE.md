## Project Snapshot
EnsemFormer trains CycloFormer for cyclic peptide membrane permeability regression from multi-conformer 3D molecular inputs.

Current model flow:
```
N conformers -> shared GNN backbone -> conformer tokens (B, N_conf, d_gnn)
                                        |
                          Transformer over conformers (ensemble mode)
                          or masked pooling (standalone mode)
                                        |
                                 MLP head -> scalar
```

Backbone options: `egnn`, `cpmp`, `se3t`.

## Source-of-Truth Files
- `scripts/main_train.py`: training entrypoint, CLI overrides, DDP wrapping.
- `scripts/preprocess_trajectories.py`: offline cache builder from trajectory PDBs.
- `scripts/prepare_dataset.py`: CSV prep utilities (rep-frame columns).
- `src/networks/cycloformer_model.py`: `CycloFormerCore` + `CycloFormerModule`.
- `src/networks/*_backbone.py`: EGNN/CPMP/SE3T backbones.
- `src/mol_dataset.py`: cache loading, env/conformer selection, split logic, collate.
- `config/default.yaml`: canonical config schema.

## Data & Runtime Rules
- Training is cache-first. `paths.cache_file` is required and must point to a `.pt` generated by preprocessing.
- `env`, `n_conformers`, and `rep_frame_only` are runtime selection knobs in `ConformerEnsembleDataModule`.
- Representative frame handling must be strict: do not clamp indices and do not silently fall back; raise explicit errors.
- Topology (`node_feat`, `adj`, `bond_types`) is molecule-level; only `(dist, coords)` vary per conformer.

## Training Commands
```bash
python scripts/main_train.py
python scripts/main_train.py --gnn_type egnn --mode ensemble
python scripts/main_train.py --gnn_type cpmp --mode standalone
torchrun --nproc_per_node=2 scripts/main_train.py
```

DDP rule: wrap the whole `modelmodule.model` (the full `CycloFormerCore`) so every learnable parameter is synchronized.
